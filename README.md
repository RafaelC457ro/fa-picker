# fa-picker

Just a Simple Javascript ES6 module for create a Fonts Awesome Picker.

![exemple](images/fa-picker.gif)


## Code Example

```js
import faPicker from 'fa-picker'

const input = document.querySelector('input')

faPicker(input)

```

## Installation


```sh
npm install fa-picker --save

```
